# 中国百强城市2026年部门预算爬取工具

## 概述
从中国百强城市政府官网爬取指定委办局的2026年部门预算文件。

## 目标委办局（32个）
卫生健康委员会、教育局、发展和改革局、规划和自然资源局、交通运输局、科技创新局、水务局、人力资源和社会保障局、工业和信息化局、市场监督管理局、国有资产监督管理委员会、公安局、公安局交通警察局、医疗保障局、商务局、文化广电旅游体育局、生态环境局、政务服务和数据管理局、城市管理和综合执法局、退役军人事务局、宣传部、司法局、住房和建设局、建筑工务署、民政局、财政局、气象局、应急管理局、审计局、政府办公厅（室）、统计局、信访局

## 安装依赖
```bash
pip install requests beautifulsoup4 lxml openpyxl
```

## 使用方法

### 爬取全部100个城市
```bash
python scrape_budgets.py
```

### 爬取指定范围
```bash
python scrape_budgets.py --start 1 --end 10    # 爬取排名1-10
```

### 爬取指定城市
```bash
python scrape_budgets.py --city 深圳
```

## 特性
- 支持断点续爬（通过 scrape_progress.json）
- 自动适配不同城市的部门名称变体
- 自动探测无预算URL城市的预算页面
- 详细日志记录（scrape.log）
- 每个城市生成爬取汇总报告

## 数据来源
- `中国百强城市排行榜_含官网.xlsx` - 城市列表和官网地址
- `city_data.json` - 解析后的城市数据（含预算页面URL）
